Web service Beta

Currently supporting:

findall:
Example usage: 
GET http://www.example.com/findall
Example JSON response body: 
[{
	"id": 1,
	"content": "Hello, Kevin!"
 }, {
	"id": 2,
	"content": "Hello, Dan!"
}]
findbyid:
Example usage: 
GET http://www.example.com/findbyid?id=2
Example JSON response body: 
{
 "id": 2,
 "content": "Hello, Dan!"
}
findalllike:
Example usage: 
GET http://www.example.com/findalllike?like=He
Example JSON response body: 
[{
	"id": 1,
	"content": "Hello, Kevin!"
 }, {
	"id": 2,
	"content": "Hello, Dan!"
}]
addgreeting:
Example usage: 
POST http://www.example.com/addgreeting
Body:
{
 "content": "Hello, John!"
}
Example JSON response body: 
{
	"id": 5,
	"content": "Hello, John!"
}
